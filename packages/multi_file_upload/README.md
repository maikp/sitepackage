# MultiFileUpload

TYPO3 extension **brezo-it/multi-file-upload**.

This extension is intended to extend the **TYPO3 Form Framework** with **multi-file upload** capabilities.

## Requirements

- TYPO3 **13.4 LTS**

## Installation

```bash
composer require brezo-it/multi-file-upload
```

## Namespace

`BrezoIt\MultiFileUpload`

## Icon handling (TYPO3 13.4)

The extension icon is registered via `Configuration/Icons.php` and stored as SVG in
`Resources/Public/Icons/Extension.svg`. No `ext_icon.svg` is needed for TYPO3 13.4.
